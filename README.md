1.frontend 
2.backend

1.frontend

in frontend i have use ReactJS
where i use framer-motion for smooth animate
instead of normal css i have used the tailwind css which faster than odinary css
framer-motion help to best ux/ui experince for user
i faced alot of challenge while building project and i learn many new things
such as react-router-dom and anchor tags for the routes
i used the assets where gave me in mail.

2.backend

in backend i have use the express for creating server and router 
dotenv for the important files such as mongoDB url and port for server
cors for connect with frontend 
mainly i have used the mongodb as my cloud database
where i create the models and controllers 

How to run this project

this for frontend

1 copy the url or download zip file
2 install on local system
3 open the project on vs code or any other platforms
4 open terminal cd frontend
5 then npm install
6 finally npm run dev

for backend 

1 cd backend
2 npm install
3 connect your mongodb and port i have gave example.env
